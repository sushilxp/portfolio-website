Tinder for Dogs - where paws meet!

A fun matchmaking app that helps dogs find furry friends nearby for playdates,
walks, or even puppy love. Tinder for Dogs is a playful and engaging web app
designed to help dog owners connect their pets with compatible canine
companions in their local area. Inspired by popular dating apps, it features
a swipe-based interface that makes finding a doggy friend fun, easy, and intuitive.

Responsive Flexbox Pricing Table

Mondrian Painting with CSS Grid

This project is a digital recreation of Piet Mondrian's iconic abstract art style using
only HTML and CSS Grid. It captures the essence of the De Stijl movement
by arranging bold, colored rectangles within a perfectly balanced grid layout.
